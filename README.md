# tcad-scrpt-gen

## 概述

`@tongleen/tcad-script-gen` 是一个 TypeScript DSL 生成库，提供用于构建 Sentaurus TCAD 仿真脚本的声明式 API。
它分为多个独立的功能模块，目前已经实现 SDE（结构定义编辑器）和 SDevice（器件仿真）的代码生成。
